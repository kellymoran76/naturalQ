File formats

JSON — full control, supports alternatives & hints:

{
  "title": "Solar Basics",
  "category": "Engineering",
  "timerSecs": 30,
  "questions": [
    {
      "question": "What does PV stand for?",
      "answer": "Photovoltaic",
      "alternatives": ["photo voltaic"],
      "hint": "Two words"
    }
  ]
}

CSV — quick & spreadsheet-friendly. First row is a header:

question,answer,hint
What does PV stand for?,Photovoltaic,Two words
Capital of France?,Paris,

For CSV, alternatives can be added in an optional alternatives column using | to separate them.
